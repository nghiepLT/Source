insertHtml Plugin For FCKeditor:

Add Description:

1. Upload insertHtml folder to  fckeditor/editor/plugins 

2. Modify fckeditor/fckconfig.js Content:
    Increase the line of code:
		FCKConfig.Plugins.Add( 'insertHtml', 'en,zh-cn' ) ; //the FCKeditor Plugin to Insert Html Code
    
3. Add a value 'insertHtml' to FCKConfig.ToolbarSets
